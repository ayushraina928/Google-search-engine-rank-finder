# Google Search Engine Rank Finder
Helps to Find the rank of any given page on Google chrome (For top 100 pages). The project will help determine thr ranks of your page by parsing through google pages using selenium. Selenium will help us by giving HTML content of the page and we will load this content to BeautifulSoup for better structure as BeautifulSoup would organize the data into a tree structure.
<br>

# Uses
A lot of people these days start SEO based buisness and would need a software to find the ranks for their pages. This will help all those to save time and money as they won't need to find it manuelly and can run it for free.
<br>
